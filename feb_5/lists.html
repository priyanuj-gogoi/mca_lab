Steps to create an HTML file:

  1. Open Notepad.
  2. Start writing some HTML code.
  3. Press Ctrl + Shift + S (shortcut for "Save as") on the keyboard to save the current file as.
  4. In the File name field, choose a desired file name of your choice and make sure to set the file extension as .html. For example: index.html.
  5. Then, click on the Save button to save the file.
  6. And your HTML has been successfully created.
